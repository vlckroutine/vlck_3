## 1. 브랜드 철학

### 브랜드 에센스

> **"Visible. Loud. Clear. Known."**  
> VLCK는 이 네 단어의 앞 글자로 이루어진 이름입니다.  
> 우리는 고객의 브랜드가 **보이고(Visible), 울려 퍼지고(Loud), 명확하게 전달되고(Clear), 기억되도록(Known)** 합니다.

### 브랜드 포지셔닝

VLCK는 **중소기업과 소상공인을 위한 실전형 마케팅 파트너**입니다.  
화려한 포트폴리오보다 **진짜 성과**를, 복잡한 전략보다 **실행 가능한 솔루션**을 제공합니다.

### 브랜드 퍼스낼리티

| 키워드 | 의미 |
|--------|------|
| **전문적인** | 데이터와 근거로 말한다 |
| **친근한** | 어렵지 않고, 함께한다 |
| **명확한** | 복잡함을 단순하게 정리한다 |
| **실행 중심의** | 말보다 결과로 증명한다 |
| **진정성 있는** | 고객의 성공이 곧 우리의 성공이다 |

### 브랜드 약속

- 고객의 예산을 내 돈처럼 씁니다.
- 성과가 없으면 이유를 먼저 설명합니다.
- 트렌드를 쫓되, 본질을 놓치지 않습니다.

## 2. 로고 사용 규정

### 로고 구성

```
[VLCK] + 서브텍스트 "Marketing Agency"
```

- **Primary Logo** — 풀 로고 (심볼 + 워드마크 + 서브텍스트)
- **Secondary Logo** — 워드마크 단독 (VLCK)
- **Symbol Mark** — 최소 공간 적용 시 단독 사용

### 여백 규정 (Clear Space)

로고 주변에는 반드시 **로고 높이(X)의 50%** 이상의 여백을 확보합니다.

```
        ← X/2 →
   ┌─────────────────┐
   │                 │  ↑
   │    [ VLCK ]     │  X/2
   │                 │  ↓
   └─────────────────┘
```

### 최소 사용 크기

| 매체 | 최소 너비 |
|------|-----------|
| 디지털 (화면) | 80px |
| 인쇄물 | 20mm |
| 파비콘 / 아이콘 | 심볼마크 단독 사용 |

### 로고 색상 버전

| 버전 | 사용 배경 |
|------|-----------|
| **Primary (다크)** | 흰색 / 밝은 배경 |
| **Reverse (화이트)** | 어두운 배경 / 컬러 배경 |
| **Mono Black** | 흑백 인쇄물 |
| **Mono White** | 단색 어두운 인쇄물 |

## 3. 컬러 시스템

### Primary Colors

#### 🟦 VLCK Blue — 메인 브랜드 컬러
```
HEX     #1A4EFF
RGB     26, 78, 255
CMYK    90, 69, 0, 0
Pantone PMS 2728 C
```
> 신뢰, 전문성, 명확함을 상징합니다.  
> CTA 버튼, 주요 제목, 강조 요소에 사용합니다.

#### ⬛ VLCK Black — 보조 메인 컬러
```
HEX     #141414
RGB     20, 20, 20
CMYK    0, 0, 0, 92
```
> 본문 텍스트, 배경, 중량감이 필요한 요소에 사용합니다.

---

### Secondary Colors

#### 🟨 Accent Yellow — 강조 포인트
```
HEX     #FFD600
RGB     255, 214, 0
CMYK    0, 16, 100, 0
```
> 주목도가 필요한 배지, 태그, 하이라이트에 제한적으로 사용합니다.

#### 🩶 Neutral Gray — 보조 텍스트 / 배경
```
HEX     #F4F5F7   (배경용 Light Gray)
HEX     #8C8C9E   (서브 텍스트용 Mid Gray)
HEX     #3D3D4E   (보조 텍스트용 Dark Gray)
```

---

### 컬러 사용 비율

```
Primary Blue  ██████░░░░  40%
Black         ████░░░░░░  30%
White / BG    ███░░░░░░░  20%
Accent Yellow █░░░░░░░░░  10%
```

> ⚠️ Accent Yellow는 전체 디자인의 10%를 초과해서는 안 됩니다.

## 4. 타이포그래피

### 국문 서체

| 용도 | 폰트 | 굵기 |
|------|------|------|
| **헤드라인** | Pretendard | Bold (700) |
| **서브타이틀** | Pretendard | SemiBold (600) |
| **본문** | Pretendard | Regular (400) |
| **캡션 / 태그** | Pretendard | Medium (500) |

> 📎 Pretendard 다운로드: [https://github.com/orioncactus/pretendard](https://github.com/orioncactus/pretendard)

### 영문 서체

| 용도 | 폰트 | 굵기 |
|------|------|------|
| **헤드라인** | Inter | Bold (700) |
| **본문** | Inter | Regular (400) |
| **코드 / 데이터** | JetBrains Mono | Regular (400) |

### 타입 스케일

| 레벨 | 크기 | 줄간격 | 용도 |
|------|------|--------|------|
| Display | 48px / 3rem | 1.2 | 랜딩 히어로 |
| H1 | 36px / 2.25rem | 1.25 | 페이지 제목 |
| H2 | 28px / 1.75rem | 1.3 | 섹션 제목 |
| H3 | 22px / 1.375rem | 1.4 | 서브섹션 |
| Body L | 18px / 1.125rem | 1.6 | 리드 카피 |
| Body | 16px / 1rem | 1.7 | 일반 본문 |
| Caption | 13px / 0.8125rem | 1.5 | 보조 설명 |

### 타이포그래피 원칙

- 한 화면에 폰트 굵기는 최대 **3가지**를 초과하지 않습니다.
- 본문 최대 가독 너비: **660px (국문 기준 약 38자)**
- 텍스트 정렬: 기본 **좌측 정렬**, 히어로 섹션만 중앙 정렬 허용

## 5. 톤앤매너

### 브랜드 보이스 (Brand Voice)

VLCK의 목소리는 **"마케팅을 잘 아는 옆집 형/언니"** 입니다.  
전문적이지만 딱딱하지 않고, 친근하지만 가볍지 않습니다.

### 커뮤니케이션 스타일

| 상황 | 어조 | 예시 |
|------|------|------|
| 서비스 소개 | 자신감 있게, 명확하게 | "결과로 증명합니다." |
| 고객 상담 | 공감하며, 해결 중심으로 | "함께 방법을 찾아볼게요." |
| SNS 콘텐츠 | 가볍고 유익하게 | "오늘 마케팅 꿀팁 하나 드릴게요 👇" |
| 성과 보고 | 명확하고 투명하게 | "이번 달 ROAS는 420%였습니다." |
| 문제 상황 | 솔직하고 책임감 있게 | "원인을 파악했고, 이렇게 개선하겠습니다." |

### DO / DON'T

| ✅ DO | ❌ DON'T |
|-------|----------|
| 쉬운 언어로 설명한다 | 마케팅 전문 용어를 남발한다 |
| 구체적인 수치로 말한다 | "많이", "엄청", "최고"처럼 모호하게 말한다 |
| 고객 입장에서 먼저 생각한다 | 회사 자랑을 앞세운다 |
| 핵심을 먼저 말한다 | 서론을 길게 늘인다 |
| 신뢰를 쌓는 말을 한다 | 과장되거나 증명 불가한 표현을 쓴다 |

## 6. 카피라이팅 원칙

### 헤드라인 작성 원칙

1. **혜택을 먼저** — "광고를 만든다"가 아니라 "매출이 오른다"
2. **숫자를 활용** — "성과를 높인다" → "ROAS 350% 달성"
3. **타겟에게 말걸기** — "소상공인이라면 꼭 알아야 할..."
4. **질문형 활용** — "아직도 광고비만 낭비하고 계신가요?"

### 금지 표현

```
❌ "업계 최고"
❌ "압도적인 실력"
❌ "믿을 수 있는"  (→ 대신 근거를 제시할 것)
❌ "저렴한 가격에"  (→ "합리적인 비용으로" 사용)
❌ "빠른 시일 내"  (→ 구체적 기간 명시)
```

### 권장 표현

```
✅ "데이터로 확인된"
✅ "평균 [수치]% 개선"
✅ "담당자 직접 운영"
✅ "합리적인 비용으로"
✅ "[기간] 내 성과 보장"
```

## 7. SNS 콘텐츠 가이드

### 채널별 콘텐츠 전략

#### 인스타그램 (@vlck_official)
- **목적** : 브랜드 인지도 + 잠재고객 유입
- **피드** : 마케팅 팁 카드뉴스, 성과 사례, 트렌드 인사이트
- **스토리** : 일상 운영 과정, Q&A, 이벤트 안내
- **릴스** : 30~60초 마케팅 꿀팁, Before/After 사례
- **업로드 주기** : 피드 주 3회 / 스토리 매일 / 릴스 주 1회

#### 네이버 블로그
- **목적** : SEO 검색 유입 + 신뢰도 구축
- **콘텐츠** : 마케팅 가이드, 업종별 사례, 서비스 상세 소개
- **업로드 주기** : 주 2회
- **권장 분량** : 1,500자 이상, 이미지 5장 이상

#### 카카오채널
- **목적** : 기존 고객 유지 + 문의 전환
- **콘텐츠** : 마케팅 뉴스레터, 프로모션 안내, 이벤트
- **발송 주기** : 월 2~4회

---

### 비주얼 콘텐츠 규격

| 형식 | 크기 | 용도 |
|------|------|------|
| 인스타 피드 (정방형) | 1080 × 1080px | 카드뉴스, 사례 |
| 인스타 피드 (세로형) | 1080 × 1350px | 인포그래픽 |
| 인스타 스토리 / 릴스 | 1080 × 1920px | 영상, 이벤트 |
| 네이버 블로그 썸네일 | 1200 × 628px | 블로그 대표 이미지 |
| 카카오 채널 이미지 | 800 × 400px | 공지, 뉴스레터 |

### 해시태그 전략

```
# 브랜드 태그 (항상 포함)
#VLCK #브이엘씨케이 #VLCK마케팅

# 서비스 태그 (게시물 유형에 따라 선택)
#소상공인마케팅 #중소기업마케팅 #디지털마케팅
#SNS마케팅 #온라인마케팅 #마케팅대행사

# 트렌드 태그 (시즌/이슈에 따라 교체)
#마케팅꿀팁 #광고운영 #브랜딩
```

> ⚠️ 인스타그램 해시태그는 **10~15개**를 권장합니다. 30개 도배 금지.

## 8. 금지 사항

### 로고 사용 금지

```
❌ 로고 비율 변형 (늘리거나 찌그러뜨리기)
❌ 로고에 그림자, 외곽선, 글로우 효과 추가
❌ 가이드에 없는 색상으로 로고 색 변경
❌ 로고 위에 다른 요소 겹치기
❌ 저해상도 이미지 확대 사용
❌ 배경과 대비가 낮아 판독 불가한 조합 사용
```

### 컬러 사용 금지

```
❌ 가이드에 없는 임의의 컬러 추가
❌ Accent Yellow를 배경 전체에 사용
❌ 두 개 이상의 브라이트 컬러 동시 사용
```

### 카피 & 커뮤니케이션 금지

```
❌ 경쟁사를 직접 비교하거나 비하하는 표현
❌ 증명되지 않은 성과를 사실처럼 기재
❌ 법적으로 보장할 수 없는 "100% 보장" 표현
❌ 고객 정보 또는 성과 데이터를 무단으로 외부 공개
❌ 정치적, 종교적 주제 관련 게시물 작성
```
